About this website

First, I did not write the code that drives what you see on this website. I am not a website developer, so I used a template from HTML5 UP and filled in the necessary details. I did, however, build the infrastructure using Amazon Web Services. Cullancarey.com is a static AWS S3 website fronted by Cloudfront. Please see the architecture diagram below.

This website is highly available. The main website bucket replicates to the failover bucket in a different region. Access to the S3 bucket is restricted to the Cloudfront OAI. All code is available for viewing in the GitHub repository. Building cullancarey.com was a fun project and a valuable experience.

Deployment Automation

The automation for this repo's deployment utilized a Cloudformation StackSet set up for my aws organization. My aws_deployment-roles repo holds CDK code which defines the Cloudformation stack set that deploys the deployment roles to the member accounts of my organization. I have created an OIDC GitHub Actions user in my management account. The actions first assume this role and then use this role to assume the deployment roles in the member accounts. Youtube Video Generator

I was looking to improve various areas of my skillset in this project and hit all my areas of interest. I wanted to create something I could share with friends and family! I was doing some research and came across Sudeep Chauhan's blog post about automating the creation of Youtube videos, and I thought it was a great idea. At the end of the post, he specifies people are free to use it, so I wanted to see if I could tweak it and create my version of his idea. See an example video below.

This program first scrapes Reddit r/quotes for the newest SFW post on the page. It then writes that quote to a text file and, using tts, creates an audio file of the quote. Next, it will do a google image search based on the text file to retrieve relevant images for the video. After that, it will create the video file using FFmpeg. Finally, it will call the upload_video.py file provided by Youtube here. The video will appear on my Youtube channel every day at 10 am EST.

The second part of the project tweets about the videos. It retrieves to most recent upload to the Youtube channel and tweets about the video to the Twitter account I created for it. The tweets contain the video title, link to the Youtube channel, link directly to the video, and the exact time of the tweet. See an example here.
