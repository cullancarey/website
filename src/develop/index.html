About this website

First, I did not write the code that drives what you see on this website. I am not a website developer, so I used a template from HTML5 UP and filled in the necessary details. I did, however, build the infrastructure using Amazon Web Services. cullan.click is a static AWS S3 website fronted by Cloudfront. Please see the architecture diagram below.

This website is highly available. The main website bucket replicates to the failover bucket in a different region. Access to the S3 bucket is restricted to the Cloudfront OAI. All code is available for viewing in the GitHub repository. Building cullan.click was a fun project and a valuable experience.

Deployment Automation

The automation for this repo's deployment utilized a Cloudformation StackSet set up for my aws organization. My aws_deployment-roles repo holds CDK code which defines the Cloudformation stack set that deploys the deployment roles to the member accounts of my organization. I have created an OIDC GitHub Actions user in my management account. The actions first assume this role and then use this role to assume the deployment roles in the member accounts.
